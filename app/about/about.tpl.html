About

TrackMyTries is an application for tracking progress on fitness goals. I built it as a hobby in order get hands-on experience in a number of open-source JavaScript development technologies. The project consists of a RESTful API backend written in JavaScript using NodeJS, Restify and MongoDB. The front end is a single page web app using AngularJS and Bootstrap. Both projects are open-source and available on Github.
