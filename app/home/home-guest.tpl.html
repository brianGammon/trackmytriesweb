About

I am a software developer from Richmond VA, and a member of SEAL Team Physical Training. TrackMyTries is an application for tracking progress on the PRT and other fitness activities.

The project consists of a RESTful API backend written in JavaScript using NodeJS, Restify and MongoDB. The front end is a single page web app using AngularJS and Bootstrap. Both projects are open-source and available on Github.
